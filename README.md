# Kiro

This project is a monorepo managed with [Turbo](https://turbo.build/).

## Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/)
- [pnpm](https://pnpm.io/)

### Installation

Install dependencies:

```bash
pnpm install
```

### Development

Start the development server:

```bash
pnpm dev
```

### Building

Build the project:

```bash
pnpm build
```

## Structure

- `apps/`: Application source code
- `packages/`: Shared packages
